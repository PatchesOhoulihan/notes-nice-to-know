  • Only Block elements can have a height and width inlines don't have any option for that.

  • Browser find ids faster than classes. If you have the opportunity use ids instead of classes

  • Wenn keine größe angegeben dann bekommt ein element die größe von seinem Eltern

  • Wenn man einem element einen fixen px wert gibt entstehen horizontale scrollbalken

  • Elemente müssen eine realtive/Absolute größe besitzen damit man sie zentrien kann.

  • Bestpractise: Sowenig schreiben wie nötig, alle Zeichen in einer Datei übertragen werden

  • Event bubbling, wenn ein Event auf einem Kindelement eintritt kann das das event auch von allen eltern elementen gesehen werden.
